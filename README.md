# Titanic Dataset - Exploratory Data Analysis 🚢

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the famous Titanic dataset to find patterns that affected passenger survival.

## Dataset
The dataset contains passenger details like:
- Age
- Sex
- Passenger Class
- Fare
- Cabin
- Embarked
- Survival

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## EDA Steps
1. Data Cleaning
2. Handling Missing Values
3. Univariate Analysis
4. Bivariate Analysis
5. Visualization of Survival Patterns

## Key Insights
- Females had higher survival rate than males
- 1st class passengers survived more
- Children survival rate was high
